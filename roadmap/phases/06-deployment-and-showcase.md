# Phase 6: Deployment, Showcase, And Advanced Extras

## Goal

Package the project as a portfolio-ready system and add optional advanced features.

## Duration

2-4 weeks

## Learning Focus

- CI/CD workflows
- Cloud deployment basics
- Public demo and technical storytelling
- Controlled scope expansion

## Build Tasks

- [ ] Add CI pipeline (lint, test, build).
- [ ] Deploy a public demo (free tier is fine).
- [ ] Add docs: architecture notes + ADRs + runbook.
- [ ] Add one optional feature (custom domains or QR code).
- [ ] Write short case-study style project summary.

## Deliverables

- Working deployment URL.
- Recruiter/interviewer-friendly docs.
- One optional advanced feature completed.

## Exit Criteria

- A new user can use your live demo.
- You can explain system trade-offs in 5-10 minutes.

## Optional Extras (Choose At Most 2)

- Custom domains
- QR code endpoint
- GeoIP enrichment
- Multi-region simulation
- gRPC internal service boundary
